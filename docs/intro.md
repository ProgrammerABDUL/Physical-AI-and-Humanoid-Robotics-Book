# Physical AI & Humanoid Robotics Course Book

Welcome to the comprehensive course book for Physical AI & Humanoid Robotics. This book provides a complete learning path from foundational concepts to advanced implementations in robotics, covering ROS 2, simulation environments, NVIDIA Isaac, and Vision-Language-Action systems.

## Course Structure

This 13-week course is organized as follows:

- **Weeks 1–2**: Foundations of Physical AI & sensors
- **Weeks 3–5**: ROS 2 fundamentals
- **Weeks 6–7**: Simulation with Gazebo/Unity
- **Weeks 8–10**: NVIDIA Isaac ecosystem
- **Weeks 11–12**: Humanoid robot design, locomotion & manipulation
- **Week 13**: Conversational & multimodal robotics using GPT + Whisper

## Integrated RAG Chatbot

This book features an integrated RAG (Retrieval-Augmented Generation) chatbot that can answer questions about the course content. You can ask questions about any topic covered in the book and receive accurate answers based on the curriculum material.
